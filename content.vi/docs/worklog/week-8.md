# Nhật Kí Công Việc Tuần 8

## Mục tiêu

- Thiết lập tính năng xác thực người dùng an toàn cho ứng dụng web.
- Tạo API backend cốt lõi (AppSync GraphQL API) cho ứng dụng.

## Công việc

| Công việc | Ngày bắt đầu | Ngày hoàn thành | Nội dung tham khảo |
|----------|----------|----------|----------|
| Thêm và cấu hình Amazon Cognito cho chức năng đăng ký và đăng nhập người dùng  | 16/11/2025 | 17/11/2025 |  |
| Tạo AppSync GraphQL API với lược đồ (schema) cho dữ liệu Hành trình và tạo mã cần thiết. | 18/11/2025 | 19/11/2025 |  |

## Thành tựu đạt được

- Tích hợp luồng xác thực đầy đủ chức năng vào ứng dụng, cho phép người dùng đăng ký và đăng nhập.
- Triển khai GraphQL API với các mutation và query, kết nối nó với một bảng DynamoDB.
